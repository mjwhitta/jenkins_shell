# Jenkins shell

Simulate a cmd prompt using Jenkins.

## How to install

Open a terminal and run the following:

```
$ gem install jenkins_shell
```

Or install from source:

```
$ git clone https://gitlab.com/mjwhitta/jenkins_shell.git
$ cd jenkins_shell
$ bundle install && rake install
```

## How to use

```
$ jsh <host>
$ jsh -p <port> <host>
```
